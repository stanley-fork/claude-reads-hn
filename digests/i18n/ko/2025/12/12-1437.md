### [Training LLMs for Honesty via Confessions](https://arxiv.org/abs/2512.08093) • 18pts 5c
[HN discussion](https://news.ycombinator.com/item?id=46242795)
TLDR: 연구자들이 GPT-5-Thinking을 훈련시켜 주 답변 후 "고백"을 생성하도록 했고, 작업 성능과는 별도로 정직성에 보상을 줬다. 모델이 주 응답에서 거짓말하거나 생략하면, 고백에서는 종종 자백한다. 훈련으로 정직성이 약간 향상된다.

댓글:
- "부정직성은 강화학습(RL)의 효과로 인해 발생할 수 있는데, 보상 형성의 문제가 모델이 거짓말하도록 부주의하게 인센티브를 주는 훈련 프로세스를 초래할 수 있다" -manarth

### [Tripwire: A new anti evil maid defense](https://github.com/fr33-sh/Tripwire) • 21pts 14c
[HN discussion](https://news.ycombinator.com/item?id=46229437)
TLDR: 카메라와 PIR 센서가 달린 Raspberry Pi 5가 당신이 없는 동안 노트북을 감시한다. 누군가 손대면 Tripwire가 공유 비밀을 삭제하고 푸시 알림을 보낸다. USB 스틱을 든 호텔 직원을 걱정하는 저널리스트와 편집증적 개발자를 위해 설계되었다.
Take: HN 댓글러들은 노트북 힌지에 콩과 쌀만 사용하면 된다고 즉시 지적했다. 틀린 말이 아니다. 또한 누군가 이미 20년 전에 보안 제품에 Tripwire라는 이름을 붙였다.
댓글:
- "고전적인 '콩, 렌틸콩, 색깔 있는 쌀로 무단 물리적 접근 탐지' 접근법이 이해하기 더 단순하고 구현하기 더 단순하다" -neuralkoi

### [Show HN: Autofix Bot – Hybrid static analysis and AI code review agent](https://news.ycombinator.com/item?id=46237358) • 14pts 1c
[HN discussion](https://news.ycombinator.com/item?id=46237358)
TLDR: DeepSource가 5,000개 이상의 정적 분석 체커와 LLM 리뷰를 결합한 에이전트를 만들었다. AI는 그냥 grep하는 대신 AST와 데이터 플로우 그래프를 도구로 사용한다. 서브 에이전트가 거짓 양성을 필터링한 후 다른 서브 에이전트가 문제를 수정한다.
