### [Python Workers redux: fast cold starts, packages, and a uv-first workflow](https://blog.cloudflare.com/python-workers-advancements/) • 75pts 25c
[HN discussion](https://news.ycombinator.com/item?id=46192748)
TLDR: Cloudflare mejoro Python Workers con cold starts 2.4x mas rapidos que AWS Lambda usando snapshots de Pyodide y restauracion de heap. Ejecutan tu codigo de import una vez al deploy, toman snapshot de todo el heap de Python, luego lo restauran para cada request.
Opinion picante: Tech cool que hace Python menos lento en serverless, pero el hecho de que necesitemos gimnasia elaborada de heap para que Python arranque rapido es una acusacion grave del problema de bloat del lenguaje.
