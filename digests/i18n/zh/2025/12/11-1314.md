### [Helldivers 2 on-disk size 85% reduction](https://store.steampowered.com/news/app/553850/view/491583942944621371) • 63pts 46c
[HN discussion](https://news.ycombinator.com/item?id=46131518)

辣评: 没有什么比意外发布 130GB 重复文件更能说"我们什么都没基准测试"了。修复它值得肯定,但想象一下要在事后分析中解释这个的开发者。

### [Python Workers redux: fast cold starts, packages, and a uv-first workflow](https://blog.cloudflare.com/python-workers-advancements/) • 75pts 25c
[HN discussion](https://news.ycombinator.com/item?id=46192748)
TLDR: Cloudflare 使用 Pyodide 快照和堆恢复改进了 Python Workers,冷启动速度比 AWS Lambda 快 2.4 倍。他们在部署时运行你的导入代码一次,快照整个 Python 堆,然后为每个请求恢复它。
辣评: 让 Python 在 serverless 中不那么慢的酷技术,但我们需要精心设计的堆体操来让 Python 快速启动,这是对语言膨胀问题的严重控诉。
评论: "我对 Python 最大的批评之一是它的冷启动时间慢。我特别注意到当我用它作为 CLI 的脚本语言时" -randomtoast
